# QW Language

QW is a minimalist compiler front-end and infrastructure written in C++. Part of the QAOS ecosystem, it focuses on a clean, modular architecture leveraging LLVM for efficient, low-level code generation.

Other languages: [tr](README.tr.md)


## Architecture

The project is divided into three core components:

1. **Front-end:** Lexical and syntactical analysis (Lexer and Parser).
2. **Semantic Analysis:** Type checking and AST (Abstract Syntax Tree) management.
3. **Back-end:** LLVM IR code generation.
